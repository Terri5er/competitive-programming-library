 * Competitive Programming Library

A personal collection of algorithms and data structures implemented in C++ for competitive programming and informatics olympiads.

** About

This repository contains implementations of common algorithms, data structures, and problem-solving techniques that I have studied and used during my preparation for programming contests and olympiads.

The purpose of this repository is to:

* Organize algorithmic knowledge
* Maintain a personal reference library
* Improve implementation skills
* Support preparation for competitive programming contests

** Topics Covered

*** Graph Algorithms

* Depth-First Search (DFS)
* Breadth-First Search (BFS)
* Lowest Common Ancestor (LCA)
* Disjoint Set Union (DSU)
* Dijkstra's Algorithm

*** Data Structures

* Sparse Table
* Segment Tree
* Fenwick Tree

*** Number Theory

* Binary Exponentiation
* Greatest Common Divisor (GCD)
* Prime Number Algorithms

*** Dynamic Programming

* Knapsack Problems
* Longest Increasing Subsequence (LIS)
* Other Dynamic Programming Techniques

*** Algorithmic Techniques

* Binary Search
* Two Pointers
* Mo's Algorithm
* Coordinate Compression

** Technologies
